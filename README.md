## Скрипты:

#### Development
Для запуска проекта на локальной машине:
```
npm install
npm run dev
```

>После клонирования и установки зависимостей проект будет доступен на http://localhost:8080/

#### Production
Чтобы создать production сборку:
```
npm run build
npm run generate
```
